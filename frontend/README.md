# newskg-demo-web

Web interface for the news KG pipeline demo.

## Project setup

```bash
conda env create -n nlpbench-web --file env.yml
conda activate nlpbench-web
npm install
```

### Compiles and hot-reloads for development

```bash
npm run dev
```

### Compiles and minifies for production

```bash
npm run build
```
